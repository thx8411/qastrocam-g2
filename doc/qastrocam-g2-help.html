qastrocam-g2 5.0 help file

Building from sources

Qastrocam now uses cmake for building. You will need cmake 2.6.8 or higher. In order to build from source, you will also need the developpement packages for Qt3. Developement packages for libusb, cfitsio, libjpeg, libavifile and SDL are recommended. On rpm distribution, you will also need the libusb-compat package

Building binary :

Please follow these recommended commands for binary building :

>gzip -d qastrocam-g2_5.0_src.tar.gz
>tar -xvf qastrocam-g2_5.0_src.tar
>mkdir qastrocam-g2-build
>cmake ../qastrocam-g2_5.0_src 6DCMAKE_BUILD_TYPE=Release
>make
>sudo make install

In order to help us to make qastrocam-g2 better, please use the command :

>make Experimental

It will send a build report to our dashboard.

Building the source package

In order to build the source package, just type :

>make package_source

Building a distribution package

In order to build a debian or an rpm package, depending on your host distribution, just type :

>make package
