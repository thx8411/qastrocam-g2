Installation de Qastrocam
2 possibilités :
Soit vous pouvez le compiler (make, make install)
Soit vous utilisez les binaires produits à cet effet.

Les modules de Qastrocam.
Si vous ne savez pas où vous en êtes il suffit de taper dans une console "qastrocam" pour que l'aide s'affiche (en anglais :) )

-db : Permet de voir l'image provenant de la caméra
-h : Active le module d'aide à la focalisation
-M : Inverse le haut, le bas, la droite et la gauche de l'image
-a : Additionne les images (stack)
-da : Affiche l'image après l'addtion
-m : Ssimule une exposition ultra longue : Utilise le max de chaque pixel
-dm : ???
-c : Active le module de suivi de l'image
-dc : Affiche l'image après alignement
-dv point de montage : Choix du point de montage de la webcam (par défaut /dev/video0)
-pc port : Port I/O du port parallele. Defaut : 378 -> LPT1 (278 pour LPT2)
-t type : selectionne le type de telescope. Taper 'help' pour de plus amples informations
-dt point de montage : Choisi le port du telescope (defaut : /dev/ttyS0)

Exemple d'utilisation :
Pour avoir le module d'aide à la focalisation et le module d'alignement et voir l'image avec une camera branchée en /dev/video1 :

qastrocam -h -c -db -dv /dev/video1

Module de mise au point de la focalisation
Commande :
qastrocam -h

Il y a 2 fenetres, la première renseigne sur une certain "taux" de focalisation qui doit être le plus elevé possible.
Le seconde présente l'histogramme de l'image.
A chaque fois que vous aurez changé quelque chose (gain, gamma, brillance, couleur, exposition.... et SURTOUT la mise au point), il faudra réinitialiser le calcul en faisant un reset

L'utilisation est très simple.
Pour le moment, ce module fontionne bien pour la lune et les planète mais le stellaire n'est pas encore tip top.

Module de préalignement-suivi
Commande :
qastrocam -c -dc
Ainsi on peut mettre ne route le module et visualiser l'image produite (utile...)

La prise en main de ce module nécessite un peu de temps mais c'est vraiment agréable de ne pas trop encombrer son disque dur.
Vous pouvez choisir le type de fichier image tout comme dans le module principal.

Le bouton Reset permet de remettre à zéro, non pas les paramètres que vous avez mis 1/4 d'heure à trouver, mais seulement les calculs inhérants à la recherche de l'objet.
A utiliser quand l'objet est perdu.

La fenêtre "hotspot" permet de régler 2 paramètres pour la recherche du spot lumineux.
Seuil : de 1 à 100 détermine la valeur des pixels qui doivent être pris en compte. Si vous objet sature presque, ce sera proche de 100. Pour l planétaire c'est proche de 70, pour le stellaire moins.
Le seuil doit être régler de manière à ce que l'image de bouge quasiment plus.

Low : Entre 0 et 100, précise la largeur de la fenêtre sur laquelle le calcul doit être fait. Attention : plus LOW est grand plus c'est groumand en ressource !!! A manipuler avec parcimonie.

Crop : Ajuste la taille de la fenêtre de sortie. Doit contenir au moins l'objet.


Attention, si votre ordinateur n'est pas tres puissant, des freezes peuvent se produire si la valeur de LOW est trop grande. La redescendre et faire un Reset.
